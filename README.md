# OODesignPatterns
Smaple implementation of Object Oriented design patterns

## List of patterns to implement:
    * Creational
        * Abstract Factory
        * Builder
        * Factory Method
        * Prototype
        * Singleton (implemented)
    * Structural
        * Adapter
        * Bridge
        * Composite
        * Decorator
        * Facade
        * Flyweight
        * Proxy
    * Behavioural
        * Chain of Responsibility
        * Command
        * Interpreter
        * Iterator
        * Mediator
        * Memento
        * Observer
        * State
        * Strategy
        * Template Method
        * Visitor
